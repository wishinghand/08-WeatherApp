# Weather App using the Open Weather API
## Technologies Used:
- HTML
- CSS
    + Materalize.css
    + SASS
- Javascript
    + Angular
    + Jquery (for Materialize.css)
    + Gulp

## How to Run
Clone the repo, `cd` into the directory and run:
```
npm install
```

and then serve it with the command:
```
gulp serve
```
